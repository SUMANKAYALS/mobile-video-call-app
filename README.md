# 📱 JustYou - Real-Time Chat Application

JustYou is a modern **real-time chat application** built using **React Native and Expo**.  
The app allows users to send messages instantly, view online users, and interact through a smooth and responsive mobile interface.

This project demonstrates modern **mobile development practices**, including authentication, real-time communication, and scalable architecture.

---

# 🚀 Features

• 🔐 Secure user authentication  
• 💬 Real-time messaging  
• 👤 User profile system  
• 🟢 Online / Offline user status  
• 🔔 Push notifications for new messages  
• 📱 Cross-platform support (Android & iOS)  
• ⚡ Fast and responsive UI  
• 🧩 Modular and scalable project structure  

---

# 🛠 Tech Stack

## Frontend
- React Native
- Expo

- Axios

## Backend / Services
- Stream Chat API (Real-time messaging)

# ⚙️ Installation

Clone the repository:
git clone https:[//github.com/yourusername/justyou-chat-app.git](https://github.com/SUMANKAYALS/mobile-video-call-app)


Navigate to the project folder:
cd justyou


Install dependencies:
npm install


---

# ▶️ Running the App

Start the development server:
npx expo start


After running the command, you can open the app using:

• Android Emulator  
• iOS Simulator  
• Expo Go App  

---

# 🔑 Environment Variables

Create a `.env` file in the root directory.

Example:
EXPO_PUBLIC_STREAM_API_KEY=your_stream_api_key
EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_key



These environment variables are required for authentication and real-time chat functionality.
